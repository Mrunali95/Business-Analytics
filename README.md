# Prediction of Movie Success by Sentimental Analysis of Comments

The objective of this project is to use YouTube user comments about the movie trailers to predict the success of the movie. A lexicon, rule-based VADER technique have been used for the sentimental analysis of the user comments to retrieve the sentiments like positive, negative and neutral. The support vector machine (SVM) machine learning algorithm, conventional neural network and Na√Øve Bayes statistical methods have been used for the prediction of the movie success using the retrieved sentiments about the comments.
